# MDX workflow for blogging

MDX を使ったブログの執筆ワークフローについて紹介します。

## MDX とは

MDX は Markdown の中で React コンポーネントを使えるようにしたフォーマットです。

## メリット

- インタラクティブなコンテンツが作れる
- コンポーネントの再利用が可能
- TypeScript との相性が良い

## まとめ

MDX を使うことで、単なる静的なブログ以上の表現力を持ったコンテンツを作成できます。
