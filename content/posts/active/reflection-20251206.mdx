## 学んだこと

### 1. Next.js App Routerのモジュール分離

```
❌ NG: クライアントコンポーネントからfsを含むモジュールをインポート
✅ OK: ユーティリティ関数を別ファイルに分離してインポート
```

**教訓**: サーバー専用の処理（fs操作など）とクライアント共通のユーティリティは明確に分離する。

### 2. Framer Motionのアニメーション

```tsx
<AnimatePresence mode="wait">
  <motion.div
    key={activeTab}
    initial={{ opacity: 0, y: 20 }}
    animate={{ opacity: 1, y: 0 }}
    exit={{ opacity: 0, y: -20 }}
  >
    {/* コンテンツ */}
  </motion.div>
</AnimatePresence>
```

タブ切り替え時のスムーズなアニメーションを実現。

### 3. useMemoによるパフォーマンス最適化

```tsx
const filteredPosts = useMemo(() => {
  if (activeTab === "all") return posts;
  return posts.filter((post) => post.category === activeTab);
}, [posts, activeTab]);
```

不要な再計算を防ぎ、フィルタリング処理を最適化。
